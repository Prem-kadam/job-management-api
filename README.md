# job-management-api
A RESTful Job Management API built with Spring Boot. Implements CRUD operations for jobs using layered architecture (Controller, Service, and Model), Dependency Injection, Interfaces, ResponseEntity, and in-memory data storage.

# Job Management API

A RESTful Job Management API built with Spring Boot.

## Features
- Create Job
- Get All Jobs
- Get Job By ID
- Update Job
- Delete Job

## Technologies
- Java
- Spring Boot
- Maven
- REST API

## Endpoints

GET /jobs
POST /jobs
GET /jobs/{id}
PUT /jobs/{id}
DELETE /jobs/{id}
